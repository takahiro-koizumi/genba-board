Dropbox連携

クラウドのDropboxに自動保存し、PC・iPadなど複数端末から同じデータを編集できます。

初回設定（5分）：
  1. Dropbox 開発者ページを開く
  2. 「Scoped access」→「App folder」を選択、名前を入力（例：現場配置ボード）→ Create app
  3. 「Settings」タブで「Redirect URIs」に を追加
  4. 「Permissions」タブで files.content.write, files.content.read, files.metadata.read をチェック → Submit
  5. 「Settings」タブの「App key」をコピーして上に貼り付け → 保存 → 「Dropboxに接続」
iPadから使うとき：同じ App Key を入力すれば iPad の Safari でも Dropbox 認証して同じデータを編集できます。
